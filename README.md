# Duck Eat

## Project Overview

Duck Eat is a SaaS that allows restaurants to create fast, mobile-first digital menu using QRCodes focusing on performance, simplicity and scalability

## Architecture Overview

- Node.js for API for admin and write operations
- Go service for public read-only endpoints
- Shared PostgreSQL
- Redis for Cache
- S3 Uploads
